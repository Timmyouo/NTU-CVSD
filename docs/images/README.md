# README images

These filenames are referenced from the [root `README.md`](../../README.md) **Visual highlights** section. Add files with **exactly** these names so the figures render on GitHub without editing the parent README.

| Filename | Purpose |
|----------|---------|
| `bch_floorplan.png` | Final BCH decoder — post-route layout (Innovus layout viewer or KLayout on `*.gds`) |
| `convolution_io.png` | HW3 — input vs output imagery, or a Verdi waveform snapshot |

**Formats:** PNG or JPG (if you use JPG, rename the root README `src` to `.jpg` or keep `.png` and convert to PNG).

**Export:** Prefer 1400–1600 px width, then scale with `width="720"` in the README for crisp Retina display.
